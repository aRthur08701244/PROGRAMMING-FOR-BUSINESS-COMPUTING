接著，阿草做了一個可以將這些線段畫出來的函數。

import matplotlib.pyplot as plt

def plotlines(linelist, canvas=[-2, 2, -2, 2], finalize=True):
    plt.figure(figsize=(5, 5))
    plt.axis(canvas)    
    for aline in linelist:
        plt.plot([aline[0], aline[2]], [aline[1], aline[3]], linestyle='-', marker='o', color='b')

輸入輸出格式¶

系統會提供一共 20 組測試資料，每組測試資料裝在一個檔案裡。

系統提供紀錄時，每一行包含一個字串以及四個整數，依序為球隊代號$T_i$、球員號碼$N_{i}$、賽季$S_i$、打數$AB_{i}$、安打數$H_i$，以逗點分隔。
第一行輸入代表第一筆紀錄，第二行輸入代表第二筆紀錄，依此類推。如果一行的輸入為「RECORDSTOP」，則代表所有的紀錄已經輸入完畢，接下來的每一行是要進行的項目，每一筆測資中可能會要進行多個項目，且各個項目可能會重複出現，直到一行的輸入為「FUNCTIONSTOP」，則代表所有要進行的項目已經輸入完畢。 項目的輸入分成幾個選項，

  • 若要進行的項目為計算球員打擊率則輸入包含三個部分，第一個部分是正整數 $1$；第二個部分是一串正整數，是要分析的球季，以逗號分隔；第三個部分是一個正整數，是要計算打擊率的球員號碼，這三個部分中間以半形空格隔開。
  • 若要進行的項目為計算球隊打擊率則輸入包含三個部分，第一個部分是正整數 $2$；第二個部分是一串正整數，是要分析的球季，以逗號分隔；第三個部分是一個字串，是要計算打擊率的球隊代號，這三個部分中間以半形空格隔開。
  • 若要進行的項目為找出表現最佳球員則輸入包含兩個部分，第一個部分是正整數 $3$；第二個部分是一串正整數，是要分析的球季，以逗號分隔，這兩個部分中間以半形空格隔開。
  • 若要進行的項目為找出表現最佳球隊則輸入包含兩個部分，第一個部分是正整數 $4$；第二個部分是一串正整數，是要分析的球季，以逗號分隔，這兩個部分中間以半形空格隔開。

輸出時，依照項目的讀入順序輸出即可，每一個項目的輸出各自佔一行，若分析結果包含多個數字，則各數字之間以半形逗號隔開。若數字有小數點，則無條件捨去至小數點第二位再印出（印出時再無條件捨去即可，運算過程中不要） 所有測資中，不會出現不同隊伍中的球員有相同號碼的情形。
如果多名球員在某一季的打擊率相同，則選擇打數較少的那位球員，若打數仍然相同，則選擇號碼數字較小的那位球員。
如果出現多個球隊在某一季的團隊打擊率相同的情形，則選擇打數較少的那個球隊，若打數仍然相同，則選擇英文字母排序較前 ($A$ 最前，$Z$ 最後)的那個球隊。
計算打擊率時，請依照題目所敘述的公式進行計算，並無條件捨去到小數點第二位。另外請注意，若打擊率經過無條件捨去到小數點第二位後，小數點後的尾數為零，請不要印出，例如若計算出的打擊率為 $0.503$，經過無條件捨去到小數點第二位後計算出 $0.50$，這時請印出 $0.5$，而不是印出 $0.50$ ；或是計算出的打擊率為 $0.006$，經過無條件捨去到小數點第二位後計算出 $0.00$，這時請印出 $0$，而不是印出 $0.00$，否則會被 PDGOS 判定為錯誤。
這裡提供一個計算無條件捨去到小數點第二位時的 function : 將計算出的結果傳入，回傳值即為符合本次輸出要求的格式。

def chop(avg):
    avg = int(avg*100) / 100
    return avg if avg > 0 else 0
